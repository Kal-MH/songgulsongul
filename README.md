# 손글송글

캘리그라피에 최적화된 편집 기술을 탑재한 커뮤니티 애플리케이션


## 목적
- 손글씨(캘리그라피)를 취미로 하는 사용자의 편의와 활동적인 취미생활을 위해 커뮤니티 어플리케이션을 제공 한다. 
- 전문 작업을 필요로 하는 디지털 창작물의 진입장벽을 낮춘다.
